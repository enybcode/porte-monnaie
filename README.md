## Resume

Porte-Monnaie est une application de bureau realisee en JavaFX pour le BTS SIO SLAM.

Elle permet de gerer un budget personnel :

- connexion utilisateur ;
- ajout de revenus et depenses ;
- suivi du solde ;
- objectifs d'epargne ;
- statistiques ;
- export CSV et PDF.

Technologies principales : Java 17, JavaFX, SQLite, JDBC, Maven, GitHub.
